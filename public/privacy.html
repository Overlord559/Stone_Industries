This page describes how the Stone Industries marketing website handles information. The site is a static brochure-style presence. It does not include accounts, checkout, booking, or on-site forms.

The site is hosted on GitHub Pages. GitHub may process standard server and delivery logs as part of providing hosting service. Refer to GitHub's documentation for details about their platform practices.

Third-party links

The site may link to external services such as email or phone clients. Those services are governed by their own policies once you leave this site.
